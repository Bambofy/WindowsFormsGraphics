WFG About

This project extends systems.drawing by adding animation capabilities. This means that you can create things like games, visualisations and simulations using the simple System.Drawing api straight out of the box.

Contents

WFG is as simple as 2 c# files, from which you extend a class and then you're ready to go. There is also demo code within the repository folder "Demo". Here is an example of an implementation:
